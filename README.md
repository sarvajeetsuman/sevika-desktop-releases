## 🔒 About

Sevika Medical Store is a comprehensive pharmacy management system for Windows.

**Note:** This is a public releases repository. The source code is maintained in a private repository.

## 💻 System Requirements

- Windows 10/11 (64-bit)
- 4 GB RAM minimum (8 GB recommended)
- 200 MB disk space
- Internet connection for API features

## 🚀 Quick Start

1. Download the installer from [Releases](https://github.com/sarvajeetsuman/sevika-desktop-releases/releases)
2. Run the installer and follow the setup wizard
3. Launch Sevika Medical Store from Start Menu
4. Configure API settings in the application
